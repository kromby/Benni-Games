**Section 2: Track Coordinates**
Build `TRACK_COORDS` as an array of 20 objects `{ row, col }` using a perimeter walk of a 6x6 grid (rows 0-5, cols 0-5):
- Top edge (left to right): row=0, col=0,1,2,3,4 (5 spaces, indices 0-4)
- Right edge (top to bottom): row=1,2,3,4,5, col=5 (5 spaces, indices 5-9)
- Bottom edge (right to left): row=5, col=4,3,2,1,0 (5 spaces, indices 10-14)
- Left edge (bottom to top): row=4,3,2,1, col=0 (4 spaces, indices 15-18)

Wait — that gives 19 spaces. Correct to get exactly 20: use a 6x6 grid where:
- Top: row=0, col=0..5 (6 spaces, indices 0-5) — corner at index 0 and index 5
- Right: row=1..5, col=5 (5 spaces, indices 6-10) — corner at index 10
- Bottom: row=5, col=4..0 (5 spaces, indices 11-15) — corner at index 15
- Left: row=4..1, col=0 (4 spaces, indices 16-19)

Total: 6+5+5+4 = 20 spaces. Corners at indices 0, 5, 10, 15. This is the correct geometry. Update CORNER_SPACES to `[0, 5, 10, 15]` — already matches.

Build this as an IIFE that returns the array:
```javascript
var TRACK_COORDS = (function () {
  var coords = [];
  var c, r;
  for (c = 0; c <= 5; c++) coords.push({ row: 0, col: c });
  for (r = 1; r <= 5; r++) coords.push({ row: r, col: 5 });
  for (c = 4; c >= 0; c--) coords.push({ row: 5, col: c });
  for (r = 4; r >= 1; r--) coords.push({ row: r, col: 0 });
  return coords;
}());
```

Add a guard: `if (TRACK_COORDS.length !== TRACK_SIZE) { throw new Error("Track geometry error: expected " + TRACK_SIZE + " spaces, got " + TRACK_COORDS.length); }`

**Section 8: Save/Load**
Follow the exact pattern from klesstann/game.js:
```javascript
function getSerializableState() {
  return {
    cars: cars.map(function (c) {
      return { id: c.id, position: c.position, lap: c.lap, finished: c.finished, finishOrder: c.finishOrder };
    }),
    state: { phase: state.phase, round: state.round, finishCount: state.finishCount }
  };
}

function saveGame() {
  try {
    localStorage.setItem(SAVE_KEY, JSON.stringify(getSerializableState()));
  } catch (e) { }
}

function loadGame() {
  try {
    var raw = localStorage.getItem(SAVE_KEY);
    if (!raw) return;
    var data = JSON.parse(raw);
    if (data.cars && data.cars.length === 4) {
      for (var i = 0; i < 4; i++) {
        cars[i].position = data.cars[i].position || 0;
        cars[i].lap = data.cars[i].lap || 0;
        cars[i].finished = !!data.cars[i].finished;
        cars[i].finishOrder = data.cars[i].finishOrder || 0;
      }
    }
    if (data.state) {
      state.phase = data.state.phase || "racing";
      state.round = data.state.round || 0;
      state.finishCount = data.state.finishCount || 0;
    }
  } catch (e) { }
}
```
