## Spacing Scale

Declared values (multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, car circle internal padding, callout padding (2px 6px uses 4px unit) |
| sm | 8px | Compact element spacing — standings row gaps, header inline gaps |
| md | 16px | Default element spacing — panel padding, roll button vertical padding |
| lg | 24px | Section padding — standings panel padding, results screen padding |
| xl | 32px | Layout gaps — gap between track board and standings sidebar |
| 2xl | 48px | Major section breaks — not used in Phase 1 |
| 3xl | 64px | Page-level spacing — not used in Phase 1 |

Exceptions:
- Track board gap between cells: 0px (cells flush — no gap between spaces, cleaner track board appearance with borders providing visual separation)
- Touch target minimum for Roll button: 44px height (mobile accessibility for kids — fingers larger than adults)
- Car circle within space: 60% of cell width (single car), 40% per circle (quadrant split)

Source: klesstann `style.css` gap/padding patterns; 44px touch target is mobile standard for young users.

| Role | Size | Weight | Line Height |
|------|------|--------|-------------|
| Body / standings text | 16px (1rem) | 400 | 1.5 |
| Label / dice result / callout | 14px (0.875rem) | 400 | 1.4 |
| Heading / panel titles / position badge | 20px (1.25rem) | 700 | 1.2 |
| Display / Roll button | 24px (1.5rem) | 700 | 1.0 |

Notes:
- Minimum rendered font size is 14px — audience is kids 7-13, nothing smaller
- Roll button uses display size (24px) to make the primary action unmissable on mobile
- Lap counter in header uses heading size (20px, weight 700) — always-visible requirement per RACEUI-02
- Position badges (1st/2nd/3rd/4th) use heading size (20px, weight 700) for at-a-glance reading

Source: CONTEXT.md constraint ("large text, clear visuals") + CLAUDE.md audience constraint (kids 7-13).

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #0d0d0d | Page background |
| Secondary (30%) | #1a1a1a | Track board background, standings panel, header bar, results screen |
| Accent (10%) | #e88a1a | Roll button background, corner space highlight tint, "1st place" badge |
| Destructive | #f44336 | Restart/reset confirmation only |

Accent reserved for: Roll button, corner space border/tint, 1st-place position badge.
Accent is NOT used for: general borders, car circles, lap counter background, regular standings rows.

Car circle colors (fixed, not part of accent system):
- Player: #e74c3c (red)
- AI 1: #3498db (blue)
- AI 2: #2ecc71(green)
- AI 3: #f39c12 (yellow/orange)

These four colors are high-contrast against the #0d0d0d background and against each other.
Source: RESEARCH.md Pattern 2 — car color values verified against existing klesstann dark theme.

Corner space visual treatment:
- Border: #e6b800 (yellow-gold, distinct from car colors and accent)
- Background tint: rgba(230, 184, 0, 0.15)
- This color is used ONLY for corner spaces — not reused elsewhere

Source: RESEARCH.md CSS Grid Track Board example.

Additional semantic colors (not accent):
- Callout background: rgba(0, 0, 0, 0.85) with white text — matches klesstann tooltip pattern
- Border/divider: #333333 — matches klesstann `--border` token

## Component Inventory

All components are hand-rolled HTML/CSS (no library). Listed for executor reference.

### Track Board
- Element: `<main id="track-board">`
- Layout: CSS Grid, `grid-template-columns: repeat(6, 1fr)`, `grid-template-rows: repeat(6, 1fr)`
- Size: `min(80vw, 480px)` width, `aspect-ratio: 1` (square)
- Only 20 perimeter cells rendered as `.track-space` divs using explicit `grid-row` / `grid-column` placement
- Interior 16 cells are empty grid area — no DOM nodes rendered there

### Track Space
- Element: `<div class="track-space">`
- Default: `border: 2px solid #333`, `border-radius: 4px`, `position: relative`
- Corner variant: `.corner` class — `border-color: #e6b800`, `background: rgba(230,184,0,0.15)`
- Start/finish space (index 0): additional `.start` class — checkered pattern via CSS background or text label
- Inner layout: CSS Grid `grid-template-columns: 1fr 1fr`, `grid-template-rows: 1fr 1fr` for quadrant split

### Car Circle
- Element: `<div class="car-circle">` inside `.track-space`
- Single car: `grid-column: 1 / -1; grid-row: 1 / -1; width: 70%`
- Two cars: each takes one quadrant; width 60%
- Three cars: quadrants 1-3; fourth quadrant empty
- Four cars: all quadrants; width 55%
- `border-radius: 50%`, `aspect-ratio: 1`, background set to car color
- Player car gets `border: 2px solid #ffffff` to distinguish from AI cars

### Corner Callout
- Element: `<div class="callout">` appended to `.track-space.corner`
- Position: `position: absolute; bottom: 100%; left: 50%; transform: translateX(-50%)`
- Style: `background: rgba(0,0,0,0.85); color: #f5f5f5; font-size: 14px; padding: 2px 8px; border-radius: 4px; white-space: nowrap; pointer-events: none`
- Animation: CSS `@keyframes callout-fade` — 1.5s, fade out + rise 12px
- JS removes node after 1600ms via `setTimeout`

### Roll Button
- Element: `<button id="roll-btn">`
- Active state: `background: #e88a1a; color: #0d0d0d; font-size: 24px; font-weight: 700; padding: 12px 32px; border-radius: 12px; border: none; cursor: pointer; min-height: 44px; width: 100%`
- Disabled state (while resolving or race finished): `opacity: 0.4; cursor: not-allowed`
- Hover: `transform: scale(1.04); opacity: 0.92`
- Label: "Rúlla!" (Icelandic for "Roll!")

### Dice Result Display
- Element: `<div id="dice-result">`
- Appears below Roll button after each roll
- Shows per-car roll summary: car color swatch + roll value
- Font: 14px, weight 400
- Cleared on next roll

### Standings Panel
- Element: `<aside id="standings-panel">`
- Always visible, updates after each round
- Four rows, one per racer, sorted 1st–4th
- Each row: position badge (20px bold) + car color circle (16px) + car label ("Þú" for player, "A1"/"A2"/"A3" for AI) + lap indicator ("Hring 1/2")
- Lap indicator color: #999 (dim text)

### Header Bar
- Element: `<header id="top-bar">`
- Contents: back link ("← Til baka") left-aligned, game title ("Klesstinn Rally") centered, lap counter right-aligned
- Lap counter: "Hring 1 af 2" — 20px, weight 700, color #f5f5f5
- Background: #1a1a1a, border-radius 12px, border 1px solid #333

### Results Screen
- Element: `<div id="results-screen" class="hidden">`
- Appears over/replaces race view when `state.phase === "finished"`
- Shows: title "Lokaniðurstaðan" (Results), ordered list of finishers with position + car color + prize money
- Prize display: position colored with finish order (1st gets accent color #e88a1a)
- Bottom action: "Keppinn aftur!" button (same style as Roll button) — Phase 1 placeholder, Phase 3 wires this fully
- Background: #1a1a1a, centered, padding 24px, border-radius 12px

## Interaction Contract

### Roll Button States

| State | Visual | Behavior |
|-------|--------|----------|
| Active (phase: racing) | Full opacity, accent background | Click/tap triggers `onRollClick()` |
| Resolving | Disabled appearance (opacity 0.4) | Not clickable during synchronous resolution (guard clause) |
| Race over (phase: finished) | Hidden or replaced by results screen | Roll button not shown |

### Corner Callout Trigger

- Fires when a car lands on or passes through a corner space during move resolution
- Text: "Hreint horn!" (Clean corner! — tire die 4, no penalty) or "Hægt! -N skref" (Slowed! -N steps — for penalties 1 or 2)
- One callout per corner per round maximum — if multiple cars hit same corner, last callout wins (replace, not stack)
- Duration: 1.5s fade, removed from DOM at 1600ms

### Position Update Timing

- Standings panel re-renders after all 4 cars resolve movement in a round
- No partial updates mid-round (all-or-nothing render)

### Race Finish Transition

- When `state.phase` changes to "finished" after `resolveRound()` returns:
  1. `renderAll()` runs one final time (standings show final positions)
  2. Results screen element has `.hidden` class removed
  3. Roll button is hidden or replaced by "Keppinn aftur!" button

### Restart (Phase 1 placeholder)

- "Keppinn aftur!" on results screen reloads the page (`window.location.reload()`) — Phase 3 replaces this with proper state reset
- No confirmation dialog needed (not destructive — player has already seen results)

## Copywriting Contract

All text in Icelandic (consistent with all existing games in codebase — `lang="is"` HTML attribute).

| Element | Copy |
|---------|------|
| Primary CTA (Roll button) | Rúlla! |
| Secondary CTA (Race again) | Keppinn aftur! |
| Back link | ← Til baka |
| Page title | Klesstinn Rally |
| Lap counter format | Hring {N} af 2 |
| Standings panel heading | Staða |
| Player car label | Þú |
| AI car labels | A1, A2, A3 |
| Results screen heading | Lokaniðurstaðan |
| Position format | {N}. sæti |
| Prize money format | {N} kr. |
| Corner callout — clean | Hreint horn! |
| Corner callout — penalized | Hægt! -{N} skref |
| Empty state (no cars rendered yet) | not applicable — initial render always shows all 4 cars at start |
| Error state | not applicable — Phase 1 has no user input that can error; dice are automatic |
| Destructive action — none in Phase 1 | n/a |

Notes:
- "Þú" (you) identifies the player car across header, standings, and results
- Prize money shown as integer with "kr." suffix — no decimal places
- Results screen shows all 4 positions even if race ended at first finisher (remaining cars get positional prize money calculated from their standings at race end)

Source: Language decision — CONTEXT.md "Claude's Discretion" (Icelandic recommended, consistent with all existing games).
