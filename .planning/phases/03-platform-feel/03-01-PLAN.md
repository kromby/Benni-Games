|-----------|----------|-----------|-------------|-----------------|
| T-03-01 | Information Disclosure | Google Fonts CDN link | accept | Font request sends standard HTTP headers (referer, IP) to Google. No PII, no auth tokens. Standard risk for any CDN-hosted font — accepted for all web properties using Google Fonts. |
| T-03-02 | Tampering | Google Fonts CSS response | accept | CDN could theoretically serve malicious CSS. Risk is industry-standard low — Google Fonts is used by millions of sites. No JS execution vector from CSS-only link. Subresource Integrity (SRI) not supported by Google Fonts dynamic CSS URLs. |
| T-03-03 | Denial of Service | Google Fonts CDN unavailability | mitigate | `display=swap` in font URL ensures text renders in fallback font (monospace) if CDN is slow or unavailable. Game remains fully playable without the pixel font. |
| T-03-04 | Tampering | Image file paths in renderTrack | accept | Image paths are hardcoded from a fixed array (`["red","blue","green","yellow"]`), not derived from user input. No path traversal or injection vector. |
